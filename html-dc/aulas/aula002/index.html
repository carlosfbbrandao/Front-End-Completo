Tags para textos

A tag <p> é utilizada para a colocação de paragrafos

A tag <p> (parágrafo) atua como bloco, isso significa que a medida que vamos acrescentando tags <p> vamos precebendo que cada uma delas é um paragrafo separado do outro criando vários blocos, diferente da tag <span>.

A tag <span> que também podemos utilizar para a colocação de paragrafos. A maior diferença entre elas é que a tag <span> é da categoria inline, que independente de quanta tags forem colocadas, elas entrarão uma do lado da outra. Ela é muito usada quando precisamos marcar uma parte dentro de um texto.
