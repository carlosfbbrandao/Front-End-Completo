Colocando links no site

Para colocar links, vamos utilizar a tag <a> e em seu atributo 'href' colocar o endereço para onde o link irá direcionar. Este link pode levar para uma página externa a página atual ou para uma página interna do site.

Para páginas extenas, alem do atributo 'href' vamos utilizar também o atributo 'target' e em seu valor colocamos a opção '_blank' para que seja aberta em outra página, sem sobrepor a atual.

Outro atributo que usamos também é o 'rel', que indica qual a natureza do link, e é muito importante para os sites de busca. Ele possue os seguintes valores: 'next/prev' - que seria para indicar uma página proxima ou anterior a atual; 'first/last' - que seria a primeir ou a ultima página da sequencia atual; 'external' - que seria quando o link direciona para uma página externa ao site atual; 'author' - que seria um link para uma página que descreve o autor ou fornece uma maneira de entrar em contato com ele.
